# Prompts Package

This folder contains system prompts and orchestration policies for the GDT agent system.

- `supervisor-agent.system.md`

## Guidelines

- Keep prompts modular and role-specific.
- Store shared constraints in policy files.
- Version prompts alongside product changes.
- Document major prompt edits in PR descriptions.
